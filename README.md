# ResiRanker
Free Residency Ranker tool based on 9 universal parameters
A simple, interactive tool to help med students stop "gut-feeling" their rank list and start using data. This app lets you set your own priorities, score your programs 1-5, and see a ranked list and visual breakdown of where your programs actually stand.

The tool focuses on these specific factors:
Resident Happiness
Case Exposure
Schedule
Fellowship Match Strength
Location Fit
Faculty Culture/Feedback
Salary vs. Cost of Living
Research Support
Program Reputation

Local Setup
If you want to run this on your own machine:

Install the requirements: pip install streamlit pandas plotly

Run the script: streamlit run app.py
